# SQL Project - Property Management System

##  Description
This project is a SQL-based database system for managing rental properties. It helps store and manage data related to properties, owners, tenants, rental agreements, and payments.

##  Technologies Used
- SQL (MySQL / SQL Server)
- Database Design Concepts

##  Files Included
- property_management.sql → Contains database schema and queries

##  Database Features
- Property listings management  
- Owner and tenant records  
- Rental agreement tracking  
- Payment records handling  
- Maintenance request structure  

## Purpose
This project was created for educational purposes to practice database design and SQL queries.
